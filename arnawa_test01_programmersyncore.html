1. Framework digunakan untuk mempermudah dan mempercepat penggunaan pada bahasa pemograman. Setiap bahasa pemograman memiliki framework karena dapat mempercepat dan mempermudah dalam pembangunan sebuah aplikasi atau website karena dalam framework sudah memiliki model seperti MVC atau komponen yang biasanya dibutuhkan. 2. If else berjalana untuk mengesekusi program jika kondisi terpenuhi contohnya jika suatu table tidak kosong (if) jalan kan program menampilkan data tersebut dan jika kosong (else) tampilkan text table kosong. While merupakan perulangan yang akan berjalan ketika kondisi terpenuhi contohnya jika ingin mengambil data dari database yang ada sampai habis. While do merupakan perulangan yang sama dengan while tetapi kode dari do harus dijalankan minimal 1 kali contohnya jika ingin memasukkan data pada table, kode dari do digunakan untuk mengetahui apakah inputan kosong jika tidak kosong while akan berjalan. 3. Constructor adalah sebuah method yang berjalan otomatis ketika object dibuat. Method merupakan serangkaian instruksi yang digunakan untuk melakukan tugas tertentu pada suatu objek. Class adalah suatu cetakan untuk membuat objek dengan karakteristik dan perilaku yang sama. Variabel adalah tempat untuk menyimpan data. Parameter merupakan sebuah nilai yang diberikan ke dalam method untuk melakukan operasi tertentu. Function adalah blok kode yang dapat dipanggil dan menjalankan operasi yang tertentu dan outputnya tergantung parameter yang diberikan. Procedure adalah serangkaian instruksi yang dijalankan untuk menyelesaikan suatu tugas tanpa mengembalikan nilai apapun. 4. Post adalah method HTTP untuk mengirimkan data ke URL sedangkan method GET digunakan untuk menampilkan data dari URL 5. Web service memungkinkan aplikasi untuk berbagi data dan fungsionalitas dengan sistem lain, bahkan jika sistem tersebut dibuat menggunakan bahasa pemrograman yang berbeda. Pada framework Laravel dapat membuat HTTP request dengan get(), post(), put(), dan delete() untuk melakukan operasi CRUD (Create, Read, Update, Delete) pada data melalui web service. 6.MVC adalah pendekatan dalam pengembangan aplikasi web yang memisahkan antara tampilan, data, dan logika aplikasi menjadi tiga bagian yang berbeda, yaitu Model, View, dan Controller. Sedangkan HMVC adalah pengembangan dari pendekatan MVC yang digunakan untuk mengatasi kompleksitas dalam pengembangan aplikasi web yang besar dan kompleks. 7. CURL adalah sebuah library untuk melakukan request HTTP dan transfer data melalui berbagai protokol seperti HTTP, FTP, dan lainnya. Dengan menggunakan CURL, developer bisa membuat koneksi ke server dan mengirim permintaan HTTP seperti GET, POST, PUT, DELETE, dan lainnya. 8. menggunakan syntax echo $variabelArray[$i] dimana variabelArray merupakan variabel array dan i adalah index dari elemen pada array yang ingin kita tampilkan. Fungsi echo digunakan untuk menampilkan nilai pada layar 9. Perbedaan HTTP dan HTTPS berada pada keamanannya dimana HTTPS melakukan enkripsi pada koneksi antara browser dengan server, sedangkan HTTP masih belum melakukan enkripsi. 10. API adalah kependekan dari Application Programming Interface, yaitu kumpulan aturan, protokol, dan alat pemrograman yang digunakan untuk memungkinkan aplikasi atau sistem yang berbeda untuk berinteraksi dan bertukar data satu sama lain. API memungkinkan pengembang untuk mengambil data atau layanan dari suatu sistem atau aplikasi, menggunakan aturan dan protokol yang telah ditentukan. contoh API Facebook, API MAL, API Steam
